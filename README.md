## 🚀 Final Year Project: Private Language Learning Webapp.
I'm current building a fully offline, Dockerized language learning webapp that eliminates third-party data tracking.

### 🏗️ The Tech Stack & Architecture
* **Frontend:** Built with **Angular 20**, leveraging **Standalone Components** and the **Signals API** for high-performance reactive state management.
* **AI Pipeline (Local-Only):**
    * **STT:** Offline Speech-to-Text via **Vosk** (Multi-language models).
    * **LLM:** Large Language Model orchestration using **Gemma 3.4B** (via LM Studio API).
    * **TTS:** High-fidelity, natural voice synthesis using **Kokoro**.
* **Backend:** Modular **Flask (Python)** architecture with a **MongoDB** persistence layer.
* **Infrastructure:** Fully containerized via **Docker & Docker Compose** for one-command deployment.
